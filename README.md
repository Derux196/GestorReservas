# Gestor de Reservas

Aplicación de consola en Java para gestionar reservas de una peluquería.

## Funciones

- Agendar reservas.
- Listar reservas del día.
- Cancelar reservas.
- Ver reporte del día.
- Validar nombre, hora, servicio y cupos disponibles.

## Tecnologías usadas

- Java
- Apache NetBeans
- Aplicación de consola

## Cómo ejecutar

1. Abrir el proyecto en Apache NetBeans.
2. Ejecutar la clase `App.java`.
3. Usar el menú de consola para interactuar con el sistema.

## Estructura del proyecto

- `App.java`: contiene el método principal y el ciclo del menú.
- `Menu.java`: muestra el menú y lee la opción del usuario.
- `Validador.java`: valida nombre, hora y servicio.
- `Operaciones.java`: gestiona reservas, cancelaciones y reportes.
